# /get_poap

POAPをmintするページのURLを返します。

本番環境において、**コミュニティメンバーが実行するのはこちらのコマンドのみ**になります。

既にURLを取得したユーザーに対しては同一のURLを返します。

# /reset_user <user id>

引数で指定したユーザーのmint履歴をリセットします。（**指定ロール/指定ユーザーのみ**）

ユーザー名から絞り込んで選択します。

本番環境で特定のユーザーのmintに問題が発生した場合に使用します。
